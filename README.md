Student Profile Web Page

Lab Assignment 02 - Web Technologies and Platforms

Project Description

This project is a Student Profile Web Page designed to demonstrate the fundamental usage of HTML structure and CSS selectors. It separates content (HTML) from presentation (CSS) and adheres to strict assignment constraints (no Flexbox, Grid, or JavaScript).

Technologies Used

HTML5: Used semantic elements.
CSS3: Used an external stylesheet.
Editor: VS Code.
